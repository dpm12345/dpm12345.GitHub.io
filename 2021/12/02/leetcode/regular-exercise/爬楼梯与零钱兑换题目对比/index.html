爬楼梯

零钱兑换 II

建立在做过题目的基础上，我们重新审视这几道题

  1. 对于爬楼梯，由于一次只能爬一阶或两阶，故dp[i]=dp[i−1]+dp[i−2]dp[i]=dp[i-1]+dp[i-2]dp[i]=dp[i−1]+dp[i−2]
    对于零钱兑换 II，其限制因素为零钱的面额，故当i>=coin[j]i>=coin[j]i>=coin[j] 时，dp[i]+=dp[i−coin[j]]dp[i]+=dp[i-coin[j]]dp[i]+=dp[i−coin[j]]

    而现在我们将爬楼梯的限制由由只有一阶或两阶，改为一个数组step[]=1,2step[]={1,2}step[]=1,2；那么在做时，便要遍历一次step[j]step [j]step[j], 当i>=step[j]i>=step[j]i>=step[j] 时，dp[i]=dp[i]+dp[i−step[j]]dp[i]=dp[i]+dp[i-step[j]]dp[i]=dp[i]+dp[i−step[j]], 而step[j]step[j]step[j] 便与前面的减去 1、2 相同。这样不难发现，如果将每次能跨越的楼梯阶数改为一个数组里的元素，那么爬楼梯与零钱兑换 II 的解题代码就十分相似了

  2. 接下来进行更深一层的讨论

    在爬楼梯题目中，我们要计算的是到达顶端的总数，每次只能一阶或两阶，由此可知，其为 若干个 1 和 2 排列，其对顺序有要求，也就是说，最后得到是排列数的结果

    而在零钱兑换 II 题目中，因为只是记录兑换一个金额的总数，先取小面额还是先取大面额是没有区别的，那么也就是说，该题求的是组合数。

    那么，该用什么方法区别这两种呢？

    下面给出两段代码

//  amount 表示金额，n 表示硬币面额数，coin [i] 表示某个硬币

	for(int i=1;i<=amount;i++)
    {
	   for(int j=0;j<n;j++)
		  {
			  if(i>=coin[j])
			  {
				  dp[i]+=dp[i-coin[j]];
			  }
		  }
	}

				  dp[j]+=dp[j-coin[i]];
			  }

经过代码观察，不难看出，第一段代码先枚举金额，然后嵌套 coin 硬币的循环； 第二段代码先枚举 coin 硬币面额，然后再枚举金额

  1. 先看第一段代码：先枚举金额，再枚举硬币面额，也就是说每一个金额的 dp 计算都会涉及到每一种硬币，当i>=coin[j]i>=coin[j]i>=coin[j], 意味着i−coin[j]i-coin[j]i−coin[j] 后可以添加一个硬币元素coin[j]coin[j]coin[j] 等于iii, 所以dp[i]=dp[i]+dp[i−coin[j]]dp[i]=dp[i]+dp[i-coin[j]]dp[i]=dp[i]+dp[i−coin[j]], 那么不难得出比如计算金额333 时，如果硬币面额存在111 和222，那么dp[1]dp[1]dp[1] 和dp[2]dp[2]dp[2] 的值都会加到dp[3]dp[3]dp[3] 上，也，就是说首先dp[3]+=dp[2]dp[3]+=dp[2]dp[3]+=dp[2], 即2，12，12，1 情况，然后dp[3]+=dp[1]dp[3]+=dp[1]dp[3]+=dp[1], 即1，21，21，2 情况。很明显，"1,2"、"2,1" 都计算的为排列数。

  2. 再看第二段代码：先枚举硬币面额，再枚举金额，也就是说每一个硬币面额都会在一种金额内计算一次，再根据循环的方式，从下标零开始遍历，知道结束，那么不难得出，coin[i]coin[i]coin[i] 的值在dp[i]dp[i]dp[i] 的每一种情况中，出现的顺序都是固定的，那么与上面分析相比，"1,2"、"2,1" 只会出现一种，所以第二段代码计算的结果为组合数

下面进行代码实现

注：为确保爬楼梯代码无误，增加了数组 b 进行检验，若正确，则应先输出 34

#include<iostream>
#include<vector>
using namespace std;

class Solution{
public:
	int climb_stairs(int amount, vector<int>&step)
	{
		vector<int>dp(amount + 1);
		int n = step.size();
		dp[0] = 1;
		for (int i = 1; i <= amount; i++)
		{
			for (int j = 0; j < n; j++)
			{
				if (i>=step[j])
				{
					dp[i] += dp[i - step[j]];
				}
			}
		}
		return dp[amount];
	}
	int change(int amount, vector<int>&coin)
	{
		vector<int>dp(amount + 1);
		int n = coin.size();
		dp[0] = 1;   // 表示金额为零的情况，
		             // 因为零钱面额也为零，故 dp [0]=1
		for (int i = 0; i < n; i++)
		{
			for (int j = 1; j <=amount; j++)
			{
				if (j>=coin[i])
				{
					dp[j] += dp[j - coin[i]];
				}
			}
		}
		return dp[amount];
	}
};

int main()
{
	int amount = 8;
	vector<int>a(3);
	a[0] = 1; a[1] = 2; a[2] = 5;
	vector<int>b(2);
	b[0] = 1; b[1] = 2;
	Solution A;
	cout << A.climb_stairs(amount, b) << endl;      
	cout <<A.climb_stairs(amount,a)  << endl;
	cout << A.change(amount,a)<< endl;
	return 0;
}
