# Hash 函数的安全性

三个问题

问题 4.1 原像

条件： Hash 函数 h:X→Yh:\mathcal{X}\rightarrow\mathcal{Y}h:X→Y 和 y\in \mathcal

找出： x∈Xx\in\mathcal{X}x∈X 使得 h(x)=yh(x)=yh(x)=y

问题 4.3 碰撞

条件： Hash 函数 h:\mathcal{X}\rightarrow\mathcal
