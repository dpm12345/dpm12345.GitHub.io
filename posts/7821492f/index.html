# Hash 函数的安全性

三个问题

问题 4.1 原像

条件： Hash 函数 h:X→Yh:\mathcal{X}\rightarrow\mathcal{Y}h:X→Y 和 y\in \mathcal

找出： x∈Xx\in\mathcal{X}x∈X 使得 h(x)=yh(x)=yh(x)=y

给定一个 (可能) 的消息摘要 y，原像问题是问是否可找到 x 使得 h(x)=yh(x)=yh(x)=y。如果对某个给定的y∈Yy\in\mathcal{Y}y∈Y，原像问题能够解决 (即找到 x)，则(x,y)(x,y)(x,y) 是有效的。不能有效解决原像问题的 Hash 函数通常称为单项 (one-way) 的或者原像稳固的 (preimage resistant) 的

问题 4.2 第二原像

条件： Hash 函数 h:X→Yh:\mathcal{X}\rightarrow\mathcal{Y}h:X→Y 和 x\in\mathcal

找出： x′∈Xx^{'}\in\mathcal{X}x′∈X 使得 x′≠xx^{'}\not=xx′=x，并且 h(x′)=h(x)h(x^{'})=h(x)h(x′)=h(x)

给定一个消息xxx，第二原像问题问是否能找到x′≠xx^{'}\not=xx′=x，使得h(x′)=h(x)h(x^{'})=h(x)h(x′)=h(x)。要注意的是，如果问题能够得到解决，则(x′,h(x))(x^{'},h(x))(x′,h(x)) 是有效的二元组。不能有效解决第二原像问题 (即找不到x′x^{'}x′ Hash 函数通常称为第二原像稳固 (second preimage resistant)

问题 4.3 碰撞

条件： Hash 函数 h:\mathcal{X}\rightarrow\mathcal

找出： x,x′∈Xx,x^{'}\in\mathcal{X}x,x′∈X 使得 x′≠xx^{'}\not=xx′=x，并且 h(x′)=h(x)h(x^{'})=h(x)h(x′)=h(x)

碰撞问题是问是否可以找到x′≠xx^{'}\not=xx′=x，使得h(x′)=h(x)h(x^{'})=h(x)h(x′)=h(x)。对这个问题的解答并不能直接产生有效的二元组。可是，如果(x,y)(x,y)(x,y) 是有效的的二元组，并且x,x′x,x^{'}x,x′ 是碰撞问题的解，则(x′,x)(x^{'},x)(x′,x) 也是一个有效的二元组。已有各种各样的方案来避免这种情况的出现。不能有效解决碰撞问题 (即找不到) 的 Hash 函数同城成为碰撞稳固。
