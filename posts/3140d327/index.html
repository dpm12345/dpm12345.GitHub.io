# 队列的实现

该实现使用的是 C++ 的函数模板

# 队列的特点

与栈有些许类似，将数据放入队列中时，放入的位置是队尾，但弹出pop()pop()pop() 操作时，是将队头元素删去，并且队列可以通过[[[ ]]] 来访问数据，故可知实现的函数功能有以下:

  • void push (T val); // 入队

  • void pop (); // 出队

  • T back (); // 返回队尾元素

  • T front (); // 返回队首元素

  • void operator=(const Queue<T>& q); // 重载 = 运算符

  • void print (); // 打印队列

  • bool empty (); // 判断是否为空

  • int size (); // 返回队列大小

# 实现之前的思考

由上面的实现功能列表，可以清楚此时队列中的元素增删分别是在尾部和头部来进行的 如果存储数据的是数组，增加数据时还好，可以直接添加，如果是删去元素，那么为了确保队列前面都有人存在 (不为循环队列), 必须分别将各个元素慢慢地向前挪，效率较慢。注意到队列的增删操作与链表的尾插和头删相同，而链表的尾插和头删比较简单，因此 此实现采用链表来存储数据

# 结点构造

由于采用的是链表来存储数据，所以直接按照链表格式创建结点即可

template<class T>
struct node{
	T data;
	node<T>* next;
};

# 函数成员

在队列的实现中，我们要进行头删，尾插，并且可以通过[[[ ]]] 来访问数据，故需要的数据分别为头结点地址、尾结点地址 (方便进行尾插)、链表的元素个数

而函数成员即为实现功能所列

所以整体结构为：

template<class T>
class Queue{
private:
	node<T>* head;         // 头结点
	node<T>* tail;         // 尾结点
	int num;
public:
	Queue();                              // 无参构造函数
	Queue(T val);                         // 有参构造函数
	~Queue();                             // 析构函数
	Queue(const Queue<T>& q);             // 复制构造函数
	void push(T val);                     // 入队
	void pop();                           // 出队
	T back();                             // 返回队尾元素
	T front();                            // 返回队首元素
	void operator =(const Queue<T>& q);   // 重载 = 运算符
	void print();                         // 打印队列
	bool empty();                         // 判断是否为空
	int size();                           // 返回队列大小

};

# 成员函数实现

# 构造函数

在上面，拟定为两种构造函数形式，无参和有参，两种方式做的工作一样，目的是对数据进行初始化。

如果是无参，headheadhead 和tailtailtail 都应指向空，而num=0num=0num=0, 如果是有参，那么新建一个结点后，headheadhead 和tailtailtail 都指向它，num=1num=1num=1

由于涉及指针，故需要深复制，重写复制构造函数

以下为实现代码：

// 无参构造函数
template<class T>
Queue<T>::Queue()
{
	head = tail = nullptr;
	num = 0;
}

// 有参构造函数
template<class T>
Queue<T>::Queue(T val)
{
	node<T>* newnode = new node<T>;
	newnode->data = val;
	newnode->next = nullptr;
	head = tail = newnode;
	num = 1;
}

// 复制构造函数
template<class T>
Queue<T>::Queue(const Queue<T>& q)
{
	node<T>*newp = q.head;
	while (newp!= nullptr)
	{
		node<T>* newnode = new node<T>;
		newnode->data = newp->data;
		newnode->next = nullptr;
		if (head == nullptr)
		{
			head = tail = newnode;
			num = 1;
		}
		else
		{
			tail->next = newnode;
			tail = tail->next;
			num++;
		}
		newp = newp->next;
	}

}

# 析构函数

由于存储数据申请了空间，那么析构函数应当执行释放内存操作

代码如下：

template<class T>
Queue<T>::~Queue()
{
	while (head != nullptr)
	{
		node<T>* temp = head;
		head = head->next;
		delete temp;
	}
	tail = nullptr;
	num = 0;
}

# push(T val)

由于push_back()push\_back()push_back() 即为链表中的尾插，那么只需创建新的结点，存储数值，之后使当前tailtailtail 的nextnextnext 指向该结点，最后tailtailtail 指向该结点即可。需要注意的是，如果队列中无元素，那么直接将headheadhead 和tailtailtail 指向该结点即可

template<class T>
void Queue<T>::push(T val)
{
	node<T>* newnode = new node<T>;
	newnode->data = val;
	newnode->next = nullptr;
	if (head == nullptr)        // 无元素，或者可以写成 tail==nullptr 或 num==0
	{
		head = tail = newnode;
		num = 1;
	}
	else
	{
		tail->next = newnode;
		tail = tail->next;
		num++;
	}
}

# pop()

所做的操作为头删

在删之前，我们必须考虑队列为空的情况，因此可以使用assert()assert()assert() 进行判断报错

当队列中有元素时，删去头结点，那么可以先创建一个结点指针指向头结点，在删去头结点后再令 head=head->next ， num--

template<class T>
void Queue<T>::pop()
{
	assert(head);
	node<T>*temp = head;
	head = head->next;
	delete temp;     // 删去 head 也可以，只不过上面的 head 赋值需换到下面写成 head=temp->next;
	num--;
}

# back()

返回队尾元素，直接使用tailtailtail 指针即可

template<class T>
T Queue<T>::back()
{
	return tail->data;
}

# front()

返回队首元素，直接使用headheadhead 指针即可

template<class T>
T Queue<T>::front()
{
	return head->data;
}

# operator=(const Queue<T>&q)

与复制构造函数类似，但这里需要判断原 Queue 对象是否已经有队列元素

// 重载 = 运算符
template<class T>
void Queue<T>::operator =(const Queue<T>& q)
{
	if (head)             // 如果存在队列
	{
		this->~Queue();
	}
	node<T>*newp = q.head;
	while (newp != nullptr)
	{
		node<T>* newnode = new node<T>;
		newnode->data = newp->data;
		newnode->next = nullptr;
		if (head == nullptr)
		{
			head = tail = newnode;
			num = 1;
		}
		else
		{
			tail->next = newnode;
			tail = tail->next;
			num++;
		}
		newp = newp->next;
	}
}

struct node{
	T data;
	node<T>* next;

template<class T>
class Queue{
private:
	node<T>* head;         // 头结点
	node<T>* tail;         // 尾结点
	int num;
public:
	Queue();                              // 无参构造函数
	Queue(T val);                         // 有参构造函数
	~Queue();                             // 析构函数
	Queue(const Queue<T>& q);             // 复制构造函数
	void push(T val);                     // 入队

	T front();                            // 返回队首元素

	bool empty();                         // 判断是否为空
	int size();                           // 返回队列大小

};

// 无参构造函数
template<class T>
Queue<T>::Queue()
{
	head = tail = nullptr;

Queue<T>::~Queue()

template<class T>
void Queue<T>::push(T val)
{
	node<T>* newnode = new node<T>;
	newnode->data = val;
	newnode->next = nullptr;
	if (head == nullptr)
	{
		head = tail = newnode;

	{

// 出队 (头删)
template<class T>
void Queue<T>::pop()
{
	assert(head);
	node<T>*temp = head;
	head = head->next;
	delete temp;
	num--;
}

// 返回队尾元素
