丑数，其所有因数均属于所给的数字集合，如对于因数 2，3，5，

2，6，10 均为满足条件的丑数

以下为 leetcode 上丑数的相关问题

# 丑数

题目链接: 丑数

题目详情：

根据丑数因数特点，我们可以依次不断除以所给因数集的各因数，如果其为丑数，结束后其必定变为数字 1 ，如果不满足，其必不为 1

代码如下:

class Solution {
public:
    bool isUgly(int n) {
        if(n<=0)
            return false;
        int div[]={2,3,5};
        for(int i=0;i<3;i++)
        {
            while(n % div[i] == 0)
            {
                n /= div[i];
            }
        }
        return n==1;

    }
};

# 丑数 II

题目链接: 丑数 II

题目详情:

思路:

这一道题目的因数集与上题相同，但要求不同，求的是满足条件的第nnn 个丑数

仔细观察题目示例，[1,2,3,4,5,6,8,9,10,12][1,2,3,4,5,6,8,9,10,12][1,2,3,4,5,6,8,9,10,12] 为前十个丑数组成的序列，结合丑数特点，每个数字最多只有2,3,52,3,52,3,5，三个因数，也就是说我们可以利用动态规划的思想，在前面的基础上乘以因数2,3,52,3,52,3,5​，得到的数必然为满足的丑数。由于要涵盖所有的数，那么前面的数都应要乘各项因数，因此可以为不同因数各设置一个记录应该乘第几个丑数的变量，并将所得乘积记录到数组中，在取时，只需取数组中的最小值即可，取出后相应的也需要更新该因数对应的最小数

由于存在公倍数，因此在判断取出的数是哪个因数时需要对每个因素都进行判断一次，排除重复的情况

代码如下:

class Solution {
public:
    int nthUglyNumber(int n) {
        vector<int> dp(n + 1);  //dp [n] 表示第 n 个丑数
        dp[1] = 1;              // 默认第一个为 1
        int p2 = 1, p3 = 1, p5 = 1;   //p2,p3,p5 分别表示对应因数应乘的丑数下标
        for (int i = 2; i <= n; i++) {
            int num2 = dp[p2] * 2, num3 = dp[p3] * 3, num5 = dp[p5] * 5;   // 更新数组
            dp[i] = min(min(num2, num3), num5);         // 找到最小值
            if (dp[i] == num2) {                        // 判断该最小值是否由因数 2 得到
                p2++;
            }
            if (dp[i] == num3) {                        // 判断该最小值是否由因数 2 得到
                p3++;
            }
            if (dp[i] == num5) {                        // 判断该最小值是否由因数 2 得到
                p5++;
            }
        }
        return dp[n];
    }
};

# 超级丑数

题目链接: 超级丑数

题目详情：

这题与与丑数 II 十分相似，不同的是，因数集不在是固定为[2,3,5][2,3,5][2,3,5]，而是由给定的数组决定。

同样的沿用丑数 II 的思想，在丑数 II 中，首先需要存储丑数的数组 dp，同时用于实现动态规划思想，其次需要有记录各因数应乘丑数的下标和记录目前丑数集的numnumnum

因此需要的变量为：

  • dp[n]dp[n]dp[n] 存储丑数
  • point[i]point[i]point[i] 记录因数primes[i]primes[i]primes[i] 应乘的丑数下标
  • nums[i]nums[i]nums[i] 存储丑数集，其中下标iii 对应因数下标iii

代码如下:

class Solution {
public:
    int nthSuperUglyNumber(int n, vector<int>& primes) {
        vector<int>point(primes.size(),0);
		// 使用 long 防止数据溢出，因为虽然题目保证第 n 各数不会超出 32-bit，
		// 但我们在使用该方法时是找已知丑数的最小值，因此可能超过 32-bit
        vector<long>nums(primes.size(),1);
        vector<long>dp(n+1);
        for(int i=1;i<=n;i++)
        {
            long minn=INT_MAX;             // 用于找到丑数集中的最小数
            for(int j=0;j<nums.size();j++)
            {
                minn=min(minn,nums[j]);
            }
            dp[i]=minn;                   // 放入存储丑数的数组中
            for(int j=0;j<nums.size();j++)
            {
                if(minn==nums[j])         // 表示目前找到的最小丑数可以有因数 primes [j] 得到
                {
                    point[j]++;           // 对应丑数小标加 1
                    nums[j]=dp[point[j]]*primes[j];     // 更新丑数集
                }
            }
        }
        return dp[n];
    }
};
